# macOS to Windows Restore Prompt

I have a private `codex-migration-payload.zip` generated by Codex Migration Kit on macOS. Please restore it on this Windows machine by:

1. Running `npx codex-migration-kit validate --payload .\codex-migration-payload.zip`.
2. Running `npx codex-migration-kit restore --payload .\codex-migration-payload.zip`.
3. Running `codex --login`.
4. Running `npx codex-migration-kit doctor` if anything fails.

Do not add project env files, workspace folders, Cursor data, or auth files to the payload.
